# UserPersistanceLayer

an API to manage user persistence layer.

## Set Up

```
$ brew install mongodb
$ clone this repo locally
$ npm install
$ mongod in another tab
$ node server.js
$ open http://localhost:8080
```

## To run the tests

```
$ ./node_modules/.bin/protractor

```
